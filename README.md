# Taller: IA para Ingenieros que Programan

Taller práctico de 3 horas donde ingenieros industriales aprenden a usar IA (Cursor + Claude) para analizar datos de producción con Python.

## Estructura del repo

```
facilitador/   # Material para quien imparte el taller
alumno/        # Material que se entrega a los participantes
```

## Requisitos

- [Cursor](https://cursor.com) instalado
- Python con `pandas` y `matplotlib`
- API Key de Claude ([console.anthropic.com](https://console.anthropic.com))

## Cómo usarlo

1. El facilitador sigue la guía en `facilitador/GUIA_FACILITADOR.md`
2. Los alumnos reciben la carpeta `alumno/` con el ejercicio y el dataset
